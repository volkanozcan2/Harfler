# İnci Harf Öğreniyor

A fun and interactive alphabet learning app for kids!

## Features

- **Interactive Alphabet**: Click on letters to see associated words and images.
- **Visual Learning**: High-quality images for each word to help with association.
- **Kid-Friendly Design**: Simple, colorful, and easy to use interface.
- **Offline Capable**: Uses local images, so no internet connection is required for the core functionality.

## Run Locally

**Prerequisites:** Node.js

1. Install dependencies:
   `npm install`

2. Run the app:
   `npm run dev`

3. Open your browser at `http://localhost:3000`
